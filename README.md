Jakarta EE  9 Demo
==================

Jakarta EE 9 Starter template to generate app.

# How to run?

```
$ mvn -DskipTests clean wildfly:run -PWildfly -Dwildfly.artifactId=wildfly-preview-dist -Dwildfly.version=22.0.0.Alpha1
```
